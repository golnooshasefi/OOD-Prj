
# Sabkino

Sabkino is a clothing e-commerce website with different features that help its users to exprience an easy shopping. Users can easily search for different products and compare them based on various properties and choose the best product to meet their needs. 
Customers finally can choose how to pay their bill.
This website is a univeristy project for OOD course.
